## Getting Started

### Installation

```bash
cd landing-page
npm install
```

### Development

```bash
npm run dev
```

The landing page will be available at `http://localhost:3001`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Framer Motion** - Animations
- **React Icons** - Icon library
- **CSS3** - Styling with custom properties

## Design Inspiration

Designs inspired by modern laundry website examples from Dribbble, featuring:
- Clean, minimalist layouts
- Professional color schemes (blues, whites, greens)
- Smooth animations and transitions
- Clear visual hierarchy
- Trust-building elements (testimonials, guarantees)

## Customization

The design uses CSS custom properties for easy theming. Main colors can be changed in `src/styles/index.css`:

```css
:root {
  --primary-color: #2563eb;
  --secondary-color: #10b981;
  --accent-color: #f59e0b;
  /* ... */
}
```
